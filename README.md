# Knockoutex

Reactive Store implemented using Knockout.js' observables. The API mimics Vuex 3.0.1

> Proof of concept

## Getting Started

### Installing

Install node packages
```
yarn install
```

Run local server
```
node server.js
```

Open proof of concept website in browser
```
http://localhost:8080/proof-of-concept/
```
